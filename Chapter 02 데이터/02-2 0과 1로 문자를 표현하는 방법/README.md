### Q18. 다음 중 문자 집합(Character Set)과 문자 인코딩(Character Encoding)의 관계로 옳지 않은 것은?

#### 3️⃣ 유니코드는 문자 집합의 한 종류이며, UTF-8, UTF-16 등은 유니코드의 인코딩 방식이다.
